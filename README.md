# WARM HEART — warmheart.studio

Premium streetwear from Malawi. Warm Heart of Africa.

## Stack
- React 18 + Vite
- Tailwind CSS
- Lucide React icons
- WhatsApp checkout flow

## Pages
- Home — hero, featured collection, brand statement, newsletter
- Shop — product grid with filter and search
- Product — full detail page with gallery, size selector, add to cart
- Lookbook — campaign photography
- About — brand story and values
- FAQ — accordion
- Contact — form + brand info
- Cart — WhatsApp checkout

## Getting Started

```bash
npm install
npm run dev
```

## Deploy
Connected to Netlify. Push to main branch to trigger deployment.

## Checkout Flow
Orders are confirmed via WhatsApp. Customers fill in name, phone, location
then are directed to WhatsApp with a pre-filled order message.
Payment via Airtel Money, TNM Mpamba, bank transfer, or cash on pickup.
